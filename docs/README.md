# Документация проекта "Геометрические фигуры"

## Общее описание решения

Данный проект содержит несколько файлов, в которых реализованы разные функции для нахождения периметров и площадей фигур.

## Структура проекта
1) `circle.py` - функции для работы с кругом
    - `area(r)`
        ```
        Возвращает площадь круга.

            Параметр:
                r (float): радиус круга

            Вощвращаемое значение:
                area (float): площадь круга

        Пример вызова: area(4)
        Вывод: 50.26548245743669
        ```

    - `perimeter(r)`
        ```
        Возвращает периметр круга.

            Параметр:
                r (float): радиус круга

            Возвращаемое значение:
                perimeter (float): периметр круга
        Пример вызова: perimeter(4)
        Вывод: 25.132741228718345
        ```

2) `rectangle.py` - функции для работы с прямоугольником
    - `area(a, b)`
        ```
        Возвращает площадь прямоугольника.

            Параметры:
                a (float): первая сторона прямоугольника
                b (float): вторая сторона прямоугольника
                (a и b - смежные стороны)

            Возвращаемое значение:
                area (float): площадь прямоугольника

        Пример вызова: area(4, 6)
        Вывод: 24
        ```

    - `perimeter(a, b)`
        ```
        Возвращает периметр прямоугольника.

            Параметры:
                a (float): первая сторона прямоугольника
                b (float): вторая сторона прямоугольника
                (a и b - смежные стороны)

            Возвращаемое значение:
                perimeter (float): периметр прямоугольника

        Пример вызова: perimeter(4, 6)
        Вывод: 20
        ```

3) `square.py` - функции для работы с квадратом
    - `area(a)`
        ```
        Возвращает площадь квадрата.

            Параметр:
                a (float): сторона квадрата

            Возвращаемое значение:
                area (float): площадь квадрата
        Пример вызова: area(5)
        Вывод: 25
        ```

    - `area(a)`
        ```
        Возвращает периметр квадрата.

            Параметр:
                a (float): сторона квадрата

            Возвращаемое значение:
                perimeter (float): периметр квадрата
        Пример вызова: perimeter(5)
        Вывод: 20
        ```

4) `triangle.py` - функции для работы с треугольником
    - `area(a, h)`
        ```
        Возвращает площадь треугольника.

            Параметры:
                a (float): сторона треугольника
                h (float): высота треугольника

            Возвращаемое значение:
                area (float): площадь треугольника

        Пример вызова: area(5, 10)
        Вывод: 25.0
        ```

    - `perimeter(a, b, c)`
        ```
        Возвращает периметр треугольника.

            Параметры:
                a (float): первая сторона треугольника
                b (float): вторая сторона треугольника
                c (float): третья сторона треугольника

            Возвращаемое значение:
                perimeter (float): периметр треугольника

        Пример вызова: perimeter(1, 2, 3)
        Вывод: 6
        ```

## Math formulas
### Area
- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²

### Perimeter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a
